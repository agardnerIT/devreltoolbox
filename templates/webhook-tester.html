📖 About

            
✅ This runs completely locally and is safe to use.

            
webhook.site is an open-source service that gives you a unique URL to receive and inspect HTTP requests in real time — useful for testing webhooks and integrations without a public server.

            
On first start the repository is cloned and Docker images are built, which can take several minutes. Subsequent starts are fast.
